## Setup
- Add `@nuxtjs/amplitude` dependency using yarn or npm to your project
- Add `@nuxtjs/amplitude` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    '@nuxtjs/amplitude',

    // With options
    ['@nuxtjs/amplitude', { /* module options */ }],
 ]
}
```
